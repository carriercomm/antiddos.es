BitTorrent Sync is a free software product developed by BitTorrent labs that allows to securely synchronize and backup files across multiple devices and platforms. BitTorrent Sync software utilizes a P2P protocol which is very effective for transferring heavy files, and is very similar to the protocol used by applications like µTorrent and BitTorrent. By its design P2P protocol requires at least one node to be on the line for you to start synchronizing or exchanging files. CoinsHost offers secure BTsync containers that are hosted on a Cloud server with 99.9% guaranteed uptime and 10 Gb/s AES-256 encrypted data transfer. With this Cloud container being on-line 24/7/365 you can securely sync, backup and exchange files across all your devices with highest speed possible.
